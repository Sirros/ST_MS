# Ant Design Pro

项目基于 [Ant Design Pro](https://pro.ant.design).

## Environment Prepare

安装项目依赖 `node_modules`:

```bash
yarn
```

### 启动项目

```bash
yarn start
```

### 项目打包

```bash
yarn build
```

### 代码格式检查

```bash
yarn lint
```

修复代码格式error：

```bash
yarn lint:fix
```

### 代码测试

```bash
yarn test
```
